Logic Gates System

Logic gates are basic building blocks of digital electronics. They control decision-making in circuits using binary inputs (0 and 1).

AND Gate

A & B → Output

Output is 1 only when both inputs are 1.

OR Gate

A + B → Output

Output is 1 if any one input is 1.

NOT Gate

A → NOT A

Reverses the input value (0 → 1, 1 → 0).
